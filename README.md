# Stream front end NextJS

## Technologies

- Next 14 App Router
- TypeScript
- [shadcn/ui](https://ui.shadcn.com/)
- [React Hook Form](https://react-hook-form.com/)
- Zod Form Validation
- Bun

## Getting Started

```bash
$ git clone URL
$ cd stream-next
$ bun install
$ bun dev
```

Open [http://localhost:3000](http://localhost:3000)
